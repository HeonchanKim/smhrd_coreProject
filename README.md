## 4. 핵심 기능
이 서비스의 핵심 기능은 이산화탄소 농도 수치가 2000ppm 이상이면 졸음운전 방지 음성을 출력합니다.  
또한 핸들에 부착되어있는 산소포화도, 체온 센서로 이상 수치 감지 시 서버로 값을 전송하고 웹에서 시각화된 그래프로 볼 수 있습니다.  
운전자가 응급 및 위급 상황 시 현재 위치를 기준으로 위도, 경도 값을 서버로 전송합니다.
